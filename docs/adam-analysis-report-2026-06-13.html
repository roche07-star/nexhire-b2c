⚡ 3. 주요 기능

3.1 이력서 분석 (핵심 기능)

  • 파일 업로드: PDF, DOCX 형식 지원
  • AI 분석: Claude API 기반 심층 분석
  • 분석 결과:
    • 직무 (Job Title)
    • 종합 요약 (Summary)
    • 핵심 강점 (Strengths) - 3개
    • 개선 필요 사항 (Improvements) - 3개
    • 핵심 키워드 (Keywords) - 상위 10개
    • 추천 커리어 경로 (Career Paths) - 3가지
  • 플랜별 차별화:
    • FREE: Claude Haiku 4.5 (빠르고 경제적, 월 3회)
    • PRO: Claude Sonnet 4.6 (심층 분석, 월 30회)
    • MANAGER: 무제한
  • 하이브리드 하네스 검증: 2단계 검증으로 정확도 향상 (70% → 90%)
  • 실시간 진행률: 업로드, 텍스트 추출, 분석 단계별 표시
  • 백그라운드 분석: 브라우저 알림으로 완료 통지

3.2 JD 적합도 분석

  • JD 템플릿 저장: 자주 사용하는 JD 미리 저장
  • 중복 방지: 동일 회사+포지션 중복 저장 차단
  • 웹 검색 통합: 회사 정보 실시간 검색 (규모, 업종, 평판)
  • 매칭 점수: 0-100점 (필수 스킬 + 우대 사항 기반)
  • 추천 판정: APPLY / CONSIDER / SKIP
  • 상세 분석:
    • 회사 인사이트 (Company Insight)
    • JD 실제 해석 (숨은 요구 역량)
    • 매칭 포인트 (3개)
    • 부족한 점 (Gaps, 2-3개)
    • 어필 전략 (Pitch Points, 3-4개)
  • 일관성 보장: 동일 입력 → 동일 점수 (±2-3%)

3.3 분석 결과 공유

  • 공유 링크 생성: 24시간 유효한 토큰 기반 URL
  • 검색 & 필터: 날짜, 키워드, 직무별 검색
  • 결과 저장: Supabase에 자동 저장

3.4 사용자 관리

  • 로그인: NextAuth (Google OAuth)
  • 플랜 관리: FREE / PRO / MANAGER
  • 사용량 제한: 월별 분석 횟수 추적
  • 쿠폰 시스템: 추가 분석 횟수 제공

3.5 랜딩 페이지

  • Hero 섹션 (메인 배너)
  • 통계 (Stats)
  • 사용 방법 (How It Works)
  • 주요 기능 (Features)
  • 페르소나 (Persona)
  • 가격 정책 (Pricing)
  • FAQ
  • CTA (Call to Action)
  • 커스텀 커서 & 스크롤 효과

3.6 기타 페이지

  • /store: 플랜 업그레이드 & 쿠폰 구매
  • /admin: 관리자 대시보드 (사용자, 플랜, 사용량 관리)
  • /privacy: 개인정보처리방침
  • /terms: 이용약관
  • /licenses: 오픈소스 라이선스 공지

8.2 Adam의 차별화 포인트

  • AI 기반 즉시 분석: 평균 2분 내 결과 (vs 전문가 3-5일)
  • 저렴한 가격: PRO 월 9,900원 (vs 사람인 50,000원/건)
  • JD 적합도 분석: 회사 정보 + 어필 전략 포함

  • 연차별 현실적 기준: 과도한 기대치 배제

8.3 개선 필요 영역

  • 브랜드 인지도: 대형 플랫폼 대비 낮음
  • 데이터 부족: 실제 채용 데이터 미보유
