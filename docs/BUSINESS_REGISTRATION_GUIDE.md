## 1. 개요

### 1.1 검토 목적

JOBIZIC 서비스의 정식 사업 개시를 위한 사업자 등록의 필요성, 절차, 주의사항 등을 종합적으로 검토하여 최적의 등록 방안을 제시한다.

### 1.2 현황

**서비스 현황:**
- 서비스명: JOBIZIC (잡이직)
- 유형: B2C SaaS (AI 이력서 분석)
- 수익 모델: 월 구독제 (PRO, EXPERT 플랜)
- 결제 시스템: 토스페이먼츠 (테스트 환경)
- 개발 진행률: 90% (결제 시스템 구축 완료)

**사업자 등록 필요성:**
1. 토스페이먼츠 정식 가맹 (실제 결제 처리)
2. 세금계산서 발행 (B2B 고객 대응)
3. 사업 신뢰도 확보
4. 법적 의무 준수

## 2. 검토 배경

### 2.1 사업 단계

**현재 단계:** MVP 완료, 베타 런칭 준비

**예상 매출:**
- 1개월차: 10명 × ₩10,000 = ₩100,000
- 3개월차: 50명 × ₩10,000 = ₩500,000
- 6개월차: 200명 × ₩10,000 = ₩2,000,000

**결론:** 소규모 초기 단계 → **개인 사업자 적합**

### 2.2 기술 스택

- Frontend: Next.js 16
- Backend: Vercel + Supabase
- 결제: 토스페이먼츠
- AI: Anthropic Claude API

**사업 분류:**
- 주업종: 소프트웨어 개발 및 공급업
- 부업종: 온라인 정보 제공 서비스업

## 3. 사업자 유형 비교

### 3.1 개인 사업자 vs 법인 사업자

| 구분 | 개인 사업자 | 법인 사업자 |
|------|-------------|-------------|
| **설립 비용** | 무료 | 50-100만원 |
| **설립 기간** | 즉시 (당일) | 2-3주 |
| **절차 복잡도** | 간단 (5분) | 복잡 (서류 다수) |
| **세금** | 종합소득세 | 법인세 |
| **세율** | 6-45% (누진세) | 10-25% (고정) |
| **장부** | 간편장부 가능 | 복식부기 필수 |
| **책임** | 무한 책임 | 유한 책임 |
| **신뢰도** | 보통 | 높음 |
| **적합 시기** | 초기 단계 | 성장 단계 |
| **연매출 기준** | ~1억원 | 1억원 이상 |

---

### 3.2 미르팀 권고사항

**✅ 개인 사업자 등록 (강력 추천)**

**근거:**
1. **비용 효율성:** 무료 등록, 낮은 운영 비용
2. **신속성:** 당일 발급 가능
3. **단순성:** 복잡한 서류 불필요
4. **충분성:** 토스페이먼츠 가맹 가능, 세금계산서 발행 가능
5. **유연성:** 향후 법인 전환 용이

**법인 전환 시점:**
- 연 매출 1억원 이상
- 투자 유치 진행 시
- 직원 5명 이상 고용 시

## 4. 미르팀 검토 의견

### 4.1 INBAE (사업 전문가)

**의견:** "개인 사업자로 즉시 시작하세요."

**이유:**
1. MVP 단계에서 법인은 과도한 투자
2. 간이과세자로 시작하면 세금 부담 최소화
3. 사업자 계좌 분리로 회계 투명성 확보
4. 통신판매업 신고로 신뢰도 보완 가능

**주의사항:**
- 간이과세자 적용 신청 필수 (연 매출 8천만원 미만)
- 사업용 계좌 개설 권장 (토스뱅크, 카카오뱅크)
- 매출 발생 시 세금 신고 기한 준수

---

### 4.2 CONAN (보안/법률 전문가)

**의견:** "법적 요구사항은 이미 충족되었습니다."

**현재 상태:**
- ✅ 개인정보 처리방침 작성
- ✅ 이용약관 작성
- ✅ 동의 시스템 구축
- ✅ 환불 정책 명시

**추가 필요:**
- 사업자 정보 공개 (Footer)
- 통신판매업 신고 (선택사항)

**법적 리스크:**
- **낮음:** 개인정보보호법 준수 완료
- **주의:** 통신판매업 신고 누락 시 과태료 (최대 1,000만원)
  → **권고:** 사업자 등록 후 2주 내 신고

---

### 4.3 DIVA (기술 총괄)

**의견:** "사업자 등록 후 5일이면 실제 결제 가능합니다."

**기술적 준비 상태:**
- ✅ 토스페이먼츠 연동 완료 (테스트 환경)
- ✅ 결제 페이지 구축
- ✅ 구독 관리 시스템 설계
- ✅ 환경변수 분리 (로컬/운영)

**사업자 등록 후 작업:**
1. 토스페이먼츠 정식 가맹 신청 (1-2일 심사)
2. 운영 키 발급 (즉시)
3. 환경변수 업데이트 (.env.local, Vercel)
4. 실제 결제 테스트
5. 정식 오픈

**예상 소요 시간:** 5영업일

---

### 4.4 DIA (UX 전문가)

**의견:** "사업자 정보 표기는 신뢰도에 필수입니다."

**UX 관점 개선사항:**
1. Footer에 사업자 정보 표기
   - 상호명, 대표자명, 사업자등록번호
   - 통신판매업신고번호
   - 주소, 이메일, 고객센터

2. 결제 페이지 개선
   - 사업자 정보 명시
   - 세금계산서 발행 안내 (B2B)

3. 마이페이지 추가
   - 구독 정보 표시
   - 결제 내역 조회
   - 세금계산서 다운로드

**기대 효과:**
- 신뢰도 30% 향상
- 전환율 20% 증가
- 고객 문의 50% 감소

3. 운영 키 발급 후

## 5. 등록 절차

### 5.1 온라인 등록 (추천)

**플랫폼:** 정부24 (https://www.gov.kr)

**소요 시간:** 5-10분

**절차:**

#### Step 1: 정부24 접속
```
1. https://www.gov.kr 접속
2. 로그인 (공동인증서 또는 간편인증)
3. 검색창에 "개인사업자 등록" 입력
4. "사업자등록 신청" 클릭
```

#### Step 2: 정보 입력

**기본 정보:**
```
상호명: JOBIZIC (잡이직)
대표자명: [본인 이름]
주민등록번호: [자동 입력]
```

**사업장 정보:**
```
사업장 주소: [자택 주소 가능]
전화번호: [연락 가능한 번호]
이메일: roche@jobizic.com
```

**업종 정보:**
```
업태: 서비스업
종목: 애플리케이션 개발 및 공급

또는

업태: 정보통신업
종목: 소프트웨어 개발 및 공급, 온라인 정보 제공업

업종 코드: 620102 (소프트웨어 개발 및 공급업)
```

**과세 유형:**
```
✅ 간이과세자 신청 (연 매출 8천만원 미만 예상)
```

#### Step 3: 제출 및 발급
```
1. 입력 내용 확인
2. 제출 버튼 클릭
3. 즉시 승인 (전산 자동 처리)
4. 사업자등록증 PDF 다운로드
```

### 5.2 세무서 방문 (대면)

**준비물:**
- 신분증
- 임대차 계약서 사본 (자가 소유 시 불필요)
- 통장 사본 (사업자 계좌 지정용, 선택사항)

**절차:**
```
1. 관할 세무서 확인
   → 홈택스 또는 국세청 콜센터 (126)

2. 방문 시간
   → 평일 09:00-18:00

3. 신청서 작성
   → 사업자등록 신청서
   → 간이과세 적용 신청서

4. 즉시 발급
   → 사업자등록증 원본 수령
```

### 6.2 통신판매업 신고

**신고 방법:**

#### Option 1: 온라인 (정부24)
```
1. 정부24 접속
2. "통신판매업 신고" 검색
3. 신청서 작성
4. 사업자등록증 사본 첨부
5. 제출
6. 2-3일 후 신고증 발급
```

#### Option 2: 방문
```
1. 관할 시청/구청 방문
2. 민원실 방문
3. 신청서 작성
4. 당일 또는 익일 발급
```

**신고 비용:** 무료

**신고증 표기:**
```
통신판매업신고번호: 2024-서울강남-00000
→ 사이트 Footer에 필수 표기
```

**미신고 시 제재:**
- 과태료: 최대 1,000만원
- 영업 정지 가능

**권고사항:** 사업자 등록 후 2주 내 신고

## 7. 사후 처리 사항

### 7.1 토스페이먼츠 정식 가맹

**현재 상태:** 테스트 키 사용 중

**절차:**

#### Step 1: 가맹 신청
```
1. 토스페이먼츠 대시보드 로그인
   https://developers.tosspayments.com

2. 우측 상단 "정식 가맹 신청" 클릭

3. 사업자 정보 입력
   - 사업자등록번호
   - 대표자명
   - 사업장 주소
   - 정산 계좌

4. 서류 업로드
   - 사업자등록증 사본 (PDF)
   - 통장 사본 (정산 계좌)
```

#### Step 2: 심사
```
소요 시간: 1-2 영업일
심사 기준:
- 사업자 정보 확인
- 서비스 적법성 검토
- 정산 계좌 확인
```

#### Step 3: 운영 키 발급
```
심사 완료 후:
1. 이메일 통지
2. 대시보드에서 운영 키 확인
   - live_ck_... (클라이언트 키)
   - live_sk_... (시크릿 키)
```

### 7.4 세금계산서 발행 시스템

**대상:** B2B 고객 (헤드헌터)

**방법:**

#### Option 1: 홈택스 수동 발행
```
1. 홈택스 로그인
```

## 8. 세금 관련

### 8.1 간이과세자 vs 일반과세자

| 구분 | 간이과세자 | 일반과세자 |
|------|-----------|-----------|
| **조건** | 연 매출 8천만원 미만 | 연 매출 8천만원 이상 |
| **부가세율** | 1.5-4% (업종별) | 10% |
| **신고 주기** | 연 1회 (1월) | 분기별 (1, 4, 7, 10월) |
| **매입세액 공제** | 불가 (간주매입율 적용) | 가능 |
| **세금계산서 발행** | 가능 (일반세금계산서) | 가능 |
| **권고 대상** | 초기 스타트업 | 성장 단계 기업 |

---

### 8.2 JOBIZIC 적용 사항

**권고:** 간이과세자 신청

**근거:**
- 초기 매출 예상: 연 2,400만원 (월 200명 × ₩10,000)
- 8천만원 미만 충족
- 낮은 세율 적용 (1.5%)
- 간단한 신고 (연 1회)

**부가세 계산 예시:**
```
월 매출: ₩2,000,000
부가세율: 1.5%
부가세: ₩2,000,000 × 1.5% = ₩30,000

연 매출: ₩24,000,000
연 부가세: ₩24,000,000 × 1.5% = ₩360,000
```

---

### 8.3 종합소득세

**세율:** 6-45% (누진세)

**계산 방식:**
```
과세표준 = 총수입금액 - 필요경비
세액 = 과세표준 × 세율 - 누진공제액
```

**세율표:**
| 과세표준 | 세율 | 누진공제액 |
|---------|------|-----------|
| 1,400만원 이하 | 6% | - |
| 1,400만원 ~ 5,000만원 | 15% | 126만원 |
| 5,000만원 ~ 8,800만원 | 24% | 576만원 |

**예시 (연 매출 2,400만원):**
```
총수입금액: ₩24,000,000
필요경비 (50% 가정): ₩12,000,000
과세표준: ₩12,000,000
세액: ₩12,000,000 × 6% = ₩720,000
```
