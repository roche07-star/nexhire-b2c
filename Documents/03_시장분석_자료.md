### 2. SAM (Serviceable Available Market)

#### 산정 기준
```
온라인 구직 활동 비율: 70%
(출처: 사람인, "구직 트렌드 리포트" 2025)

SAM = TAM × 70%
    = ₩79,800,000,000 × 0.7
    = ₩55,860,000,000 (약 559억원)
```

### 3. SOM (Serviceable Obtainable Market)

#### 3년 내 목표
```
시장 점유율 목표: 1%

SOM = SAM × 1%
    = ₩55,860,000,000 × 0.01
    = ₩558,600,000 (약 5.6억원)

5년 내 목표: 5% (약 28억원)
```

## 🔍 고객 니즈 분석

### 1. 구직자 Pain Points

#### Top 3 Pain Points
```
1위: "내 이력서가 경쟁력이 있는지 모르겠다"
     - 68% (설문 조사, N=500)
     
2위: "JD 요구사항에 내가 맞는지 불확실하다"
     - 61%
     
3위: "면접 준비 시간이 부족하다"
     - 54%
```

#### 현재 해결 방법
```
✅ 커리어 컨설팅 (30~50만원)
   - 문제: 비싸고 느림 (1~2주)
   
✅ 무료 AI 도구 (ChatGPT 등)
   - 문제: 피상적, 일관성 부족
   
✅ 지인에게 피드백 요청
   - 문제: 전문성 부족
```

#### JOBIZIC 가치 제안
```
✅ 정확도: Claude 4.5 기반 맥락 이해
✅ 속도: 10분 내 완성
✅ 가격: ₩9,900 (컨설팅 대비 1/30)
✅ 일관성: AI 기반 표준화된 분석
```

### 2. 헤드헌터 Pain Points

#### Top 3 Pain Points
```
1위: "후보자 분석에 시간이 너무 많이 걸린다"
     - 후보자 1명당 30~60분 소요
     
2위: "클라이언트 제안서 작성이 부담스럽다"
     - 제안서 1건당 2~3시간
     
3위: "정산 관리가 복잡하다"
     - 수작업 Excel, 실수 위험
```

#### JOBIZIC 가치 제안
```
✅ 시간 단축: 60분 → 10분 (1/6)
✅ 자동 제안서: 3시간 → 10분 (1/18)
✅ 정산 자동화: 클릭 한 번으로 완료
✅ 일관성: 표준화된 품질
```
