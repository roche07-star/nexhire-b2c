## 1. 사업 개요

### 1.1 사업 아이템

**JOBIZIC (조비직)** - AI 기반 이력서 분석 SaaS 플랫폼

#### 핵심 가치 제안
```
"10분이면 이력서 분석부터 면접 준비까지"
```

#### 해결하는 문제
1. **구직자**: 이력서 경쟁력 파악 어려움, JD 매칭 불확실성, 면접 준비 시간 부족
2. **헤드헌터**: 후보자 분석 시간 과다 (건당 30~60분), 일관성 부족, 정산 관리 복잡

#### 제공하는 솔루션
- Claude 4.5 기반 정확한 이력서 분석
- 10분 내 5가지 핵심 기능 제공
- 월 9,900원부터 시작하는 합리적 가격

### 1.2 사업 목표

#### 단기 목표 (6개월)
- 가입자 1,000명 확보
- MRR ₩3,000,000 달성
- FREE → PRO 전환율 10%

#### 중기 목표 (12개월)
- 가입자 5,000명 확보
- MRR ₩10,000,000 달성 (ARR ₩120M)
- B2B 고객 100개사 확보

#### 장기 목표 (24개월)
- ARR ₩500,000,000 달성
- 시리즈 A 투자 유치
- 글로벌 확장 (영어권)

### 1.3 사업 추진 배경

#### 시장 기회
- 국내 구직자 연 250만 명 (통계청, 2025)
- HR Tech 시장 연 15% 성장 (Grand View Research)
- AI 기반 SaaS 시장 연 25% 성장 (Gartner)

#### 기술 혁신
- Claude 4.5의 한국어 맥락 이해 능력
- Prompt Caching으로 비용 최적화
- 서버리스 아키텍처로 확장성 확보

#### 팀 역량
- 15년차 개발자 중심의 미르팀
- 투자 자문 전문가 합류
- HR Tech + AI 경험 보유

## 2. 시장 분석

### 2.1 시장 규모

#### TAM (Total Addressable Market)
```
국내 시장:
- 구직자: 250만 명 × ₩30,000 = ₩75,000,000,000
- 헤드헌터: 2만 명 × ₩240,000 = ₩4,800,000,000
────────────────────────────────────────────
합계: ₩79,800,000,000 (약 800억원 / 연)
```

#### SAM (Serviceable Available Market)
```
온라인 구직 활동 비율 70% 적용
= ₩55,900,000,000 (약 559억원 / 연)
```

#### SOM (Serviceable Obtainable Market)
```
목표 시장 점유율 1% (3년 내)
= ₩559,000,000 (약 5.6억원 / 연)
```

### 2.2 시장 동향

#### 성장 동인
1. **AI 기술 발전**: GPT, Claude 등 LLM 발전
2. **디지털 전환**: HR 프로세스 디지털화 가속
3. **비용 절감**: 기업의 채용 비용 절감 니즈
4. **원격 근무**: 온라인 채용 증가

#### 시장 트렌드
- **AI 도입**: HR Tech 기업의 80%가 AI 도입 계획
- **SaaS 선호**: 구독 모델 선호도 증가
- **데이터 기반**: 채용 결정의 데이터 의존도 증가

### 2.3 목표 고객

#### 1차 타겟: 개인 구직자 (B2C)
```
세분화:
- 이직 준비 재직자 (25~35세)
- IT/스타트업 분야
- 연봉 4,000만원 이상

규모: 약 50만 명
```

#### 2차 타겟: 헤드헌터 (B2B)
```

- 소규모 헤드헌팅펌 (5~20명)
- 리크루팅 에이전시

규모: 약 1만 명
```

#### 3차 타겟: 기업 (B2B2C)
```
세분화:
- 스타트업 / 중소기업
- 연 100명 이상 채용
- HR 디지털화 니즈

규모: 약 5,000개사
```

## 3. 제품 및 서비스

### 3.1 핵심 기능

#### 1. 이력서 분석
```
입력: PDF/DOCX/HWP 이력서
출력:
- 직무 적합도 점수 (0~100)
- 시장 경쟁력 점수
- 성장 가능성 점수
- 핵심 키워드 추출
- 강점 / 개선 포인트
- 커리어 방향 3가지 시나리오
```

#### 2. JD 매칭 분석
```
입력: 이력서 + JD 텍스트
출력:
- 적합도 점수 (0~100)
- 추천 레벨 (지원 권장/고려/비추천)
- 매칭 강점 / 부족한 점
- 어필 전략
```

#### 3. 이력서 자동 생성
```
입력: 원본 이력서 + JD
출력:
- JD 맞춤 재작성된 이력서
- 매칭 강점 부각
- 부족한 점 보완
- 자기소개서 최적화
```

#### 4. 면접 가이드 (EXPERT 전용)
```
입력: 이력서 + JD
출력:
- 예상 질문 10개
- 이력서 기반 모범 답변
- 역질문 3가지 (역할/도전/기대)
- 10일간 저장
```

#### 5. 채용 프로세스 관리 (헤드헌터 전용)
```
기능:
- 후보자별 진행 단계 추적 (서류→1차→2차→최종→합격)
- 프로세스 바 및 상태 표시
- 다음 액션 알림
- 진행 통계 대시보드
- 정산 기능 (매출/수수료 관리)
```

### 3.2 기술 차별점

#### AI 정확도
- Claude 4.5 기반 맥락 이해
- 한국어 특화 분석
- Tool Use로 구조화된 출력

#### 속도
- 이력서 분석: 평균 8초
- JD 매칭: 평균 5초
- 면접 가이드: 평균 12초

#### 보안
- PII 마스킹 (개인정보 보호)
- RLS (Row Level Security)
- HTTPS + Vercel 인프라
