## 📌 슬라이드 2: 문제 정의 (Problem)

### 😰 구직자의 고민
```
"내 이력서가 경쟁력이 있을까?"
"이 회사 JD에 내가 맞을까?"
"면접에서 뭘 물어볼까?"
```

### 📊 시장 현황
- 🇰🇷 국내 구직자: **연 250만 명** (통계청, 2025)
- 💼 헤드헌터: **2만 명 이상** 활동 중
- ⏰ 이력서 분석 평균 소요 시간: **30~60분 / 1건**
- 💰 헤드헌터 후보자 분석 비용: **건당 5~10만원**

### ❌ 기존 솔루션의 한계
1. **사람 기반 컨설팅**: 비싸고 (30~50만원), 느림 (1~2주)
2. **단순 키워드 매칭**: 피상적, 맥락 이해 불가
3. **헤드헌터 수작업**: 시간 소모, 일관성 부족

## 📌 슬라이드 3: 솔루션 (Solution)

### 💡 JOBIZIC = AI 이력서 분석 플랫폼

**10분이면 완성되는 5가지 기능**

1. **📊 이력서 분석**
   - 직무 적합도, 시장 경쟁력, 성장 가능성 점수
   - 핵심 키워드, 강점, 개선 포인트

2. **📋 JD 매칭 분석**
   - 적합도 점수 (0~100)
   - 매칭 강점 / 부족한 점
   - 어필 전략 제시

3. **✍️ 이력서 자동 생성**
   - JD 맞춤 이력서 재작성
   - 매칭 강점 부각, 부족한 점 보완
   - 자기소개서 최적화

4. **🎤 면접 가이드** (EXPERT 전용)
   - 예상 질문 10개 + 모범 답변
   - 역질문 3가지 제안

5. **📊 채용 프로세스 관리** (헤드헌터 전용)
   - 후보자별 진행 단계 추적
   - 다음 액션 알림
   - 정산 기능

### 🎯 핵심 차별점
```
✅ Claude 4.5 기반 정확한 분석 (키워드 매칭 X)
✅ 10분 완성 (사람 컨설팅 대비 1/100 시간)
✅ 월 9,900원~ (컨설팅 대비 1/30 비용)
✅ B2C → B2B 확장 (헤드헌터 전용 기능)
```

## 📌 슬라이드 4: 시장 규모 (Market Size)

### 📊 TAM / SAM / SOM

#### TAM (Total Addressable Market)
```
국내 구직자 250만 명 × ₩30,000 (연평균) = 750억원
+ 헤드헌터 2만 명 × ₩240,000 (연평균) = 48억원
──────────────────────────────────────────
= 798억원 (연간)
```

#### SAM (Serviceable Available Market)
```
온라인 구직 활동 비율 70% 적용
= 559억원 (연간)
```

#### SOM (Serviceable Obtainable Market)
```
목표 시장 점유율 1% (3년 내)
= 5.6억원 (연간)
```

- HR Tech 시장: **연 15% 성장** (Grand View Research, 2025)
